# 부루마블 온라인 보드게임

1982년 출시된 한국 대표 보드게임 **부루마블**을 웹으로 재현한 프로젝트입니다.
원판 40칸 보드를 충실히 구현하여 브라우저에서 2~4인이 한 화면에서 번갈아 플레이할 수 있습니다.

## 주요 기능

- **40칸 원판 보드** - 실제 부루마블 보드판을 11x11 CSS Grid로 완벽 재현
- **2~4인 로컬 핫시트** - 한 화면에서 번갈아가며 플레이
- **커스텀 플레이어** - 이름 입력, 고유 색상/토큰 자동 배정
- **주사위 시스템** - 더블 판정, 3연속 더블 감옥행 포함
- **땅 구매 & 건물 건설** - 별장/빌딩/호텔 3단계 업그레이드
- **통행료 계산** - 건물 단계별 차등 요금, 독점 보너스
- **황금열쇠 30장** - 이동, 세금, 수익, 반액대매출, 우대권, 탈출권 등
- **특수 칸** - 무인도(3턴 대기), 우주여행, 세계일주, 사회기금, 올림픽
- **서울 특수 규칙** - 건물 불가, 통행료 200만원 고정
- **이동수단** - 콩코드/퀸엘리자베스/컬럼비아호 (보유 수에 따라 통행료 배수)
- **통행료 팝업** - 타인 소유지 도착 시 금액/소유자 확인 후 지불
- **Auto Play** - 파산자가 나올 때까지 자동 진행하는 매크로 기능
- **부동산 현황 패널** - 10개 구역별 소유 상태 실시간 표시
- **보드 크기 조절** - 슬라이더로 40~110px 범위 동적 조절
- **게임 설명서** - 상세 규칙 HTML 페이지 (`/rules`)

## 프로젝트 구조

```
src/
├── app/                          # Next.js App Router
│   ├── layout.tsx                # 루트 레이아웃 (폰트, 메타데이터)
│   ├── page.tsx                  # 메인 페이지 (로비 ↔ 게임 화면 전환)
│   ├── globals.css               # 글로벌 스타일
│   └── rules/
│       └── page.tsx              # 게임 설명서 페이지
│
├── types/
│   └── game.ts                   # 타입 정의 (Tile, Player, GoldenKeyCard 등)
│
├── lib/                          # 순수 로직 & 데이터
│   ├── boardData.ts              # 40칸 보드 타일 데이터 + 그리드 좌표 매핑
│   ├── goldenKeyData.ts          # 황금열쇠 30장 카드 데이터
│   ├── diceUtils.ts              # 주사위 유틸리티 (굴림, 눈 위치)
│   └── gameEngine.ts             # 게임 엔진 (렌트 계산, 파산 처리, 카드 실행)
│
├── hooks/
│   └── useAutoPlay.ts            # Auto Play 훅 (자동 진행 로직)
│
├── store/
│   └── gameStore.ts              # Zustand 중앙 상태 관리 (게임 전체 상태 + 액션)
│
└── components/
    ├── board/                    # 보드 렌더링
    │   ├── Board.tsx             # 11x11 그리드 보드 컨테이너
    │   ├── Tile.tsx              # 개별 타일 (소유자, 건물, 토큰 표시)
    │   └── BoardCenter.tsx       # 보드 중앙 (로고, 주사위, 액션 모달)
    │
    ├── game/                     # 게임 화면
    │   ├── GameLobby.tsx         # 로비 (플레이어 수/이름 설정)
    │   ├── GameScreen.tsx        # 게임 메인 화면 (3단 레이아웃)
    │   └── GameResult.tsx        # 게임 결과 오버레이
    │
    └── ui/                       # UI 컴포넌트
        ├── DiceRoller.tsx        # 주사위 시각화
        ├── ActionModal.tsx       # 구매/건설/찬스 액션 모달
        ├── GoldenKeyCard.tsx     # 황금열쇠 카드 표시
        ├── PlayerPanel.tsx       # 플레이어 정보 패널
        ├── GameLog.tsx           # 게임 로그 (자동 스크롤)
        └── PropertyMap.tsx       # 부동산 현황 (구역별 소유 상태)
```

## 아키텍처

```
┌─────────────┐     ┌──────────────┐     ┌─────────────┐
│  Components  │────▶│  Zustand     │────▶│  Game       │
│  (React UI)  │◀────│  Store       │◀────│  Engine     │
└─────────────┘     └──────────────┘     └─────────────┘
       │                    │                    │
       │                    ▼                    ▼
       │            ┌──────────────┐     ┌─────────────┐
       └───────────▶│  Board Data  │     │  Golden Key │
                    │  (40 Tiles)  │     │  (30 Cards) │
                    └──────────────┘     └─────────────┘
```

- **Components** → UI 렌더링, 사용자 입력 처리
- **Zustand Store** → 중앙 상태 관리 (players, ownership, buildings, phase 등)
- **Game Engine** → 순수 함수 기반 게임 로직 (렌트 계산, 파산, 카드 실행)
- **Board Data** → 40칸 타일 정보 (이름, 가격, 통행료, 그리드 좌표)
- **Golden Key** → 30장 카드 데이터 (이동, 세금, 수익, 특수 효과)

## 시작하기

### 설치

```bash
git clone https://github.com/woogi8/woogi_bluemable_boardgame.git
cd woogi_bluemable_boardgame
npm install
```

### 실행

```bash
# 개발 서버
npm run dev

# 프로덕션 빌드
npm run build
npm start
```

브라우저에서 [http://localhost:3000](http://localhost:3000) 접속

## 게임 플레이

1. **로비** - 플레이어 수(2~4명) 선택, 이름 입력 (비워두면 P1, P2... 자동)
2. **주사위** - 보드 중앙 🎲 버튼 클릭
3. **행동** - 빈 땅 구매, 내 땅 건물 건설, 황금열쇠 카드 실행
4. **통행료** - 상대 소유지 도착 시 팝업으로 금액 확인 후 지불
5. **Auto Play** - 좌측 패널 `▶ Auto ON` 버튼으로 자동 진행 (파산 시 자동 종료)
6. **승리** - 마지막 생존자 또는 최다 자산 보유자
